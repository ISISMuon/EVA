Data searching

Descriptions of all functions used for database searching. Note that there are inconsistencies within the naming and the return types between different functions, so take care to read the docs carefully.

EVA.core.data_searching.get_match.search_gammas(input_peaks)

Searches for possible gamma transitions in the database at multiple energies at once.

Parameters:

input_peaks (list[list[float]]) – list of [search energy, search width] search parameters.

Return type:

list[dict]

Returns:

List of dictionaries containing matches with keys

  • isotope: the isotope which was searched for.

  • energy: energy of transition matched in database.

  • diff: difference between searched energy and match (how close the match is).

  • lifetime: lifetime of transition

  • intensity: intensity of transition.

EVA.core.data_searching.get_match.search_gammas_single_isotope(isotope)

Retrieves all gamma transitions available in the gamma database for specified isotope.

Parameters:

isotope (str) – which isotope to search for

Return type:

list[dict]

Returns:

List of dictionaries, one for each match, with keys

  • isotope (str)

  • energy (float)

  • intensity (str)

  • lifetime (str)

EVA.core.data_searching.get_match.search_gammas_single_transition(isotope, energy)

Searches in the gamma database for all transitions within 0.5% of specified energy for a given isotope.

Parameters:

Return type:

list[dict]

List of dictionaries, one for each match, with keys

EVA.core.data_searching.get_match.search_muxrays(input_peaks)

Searches for possible muonic xray transitions in the database at multiple energies at once.

NOTE: This function searches within an error range of 3x search width. if you’d only like energies within your search width, you will have to filter out matches outside the search width using the “error” key in the results.

Parameters:

input_peaks (list[list[float]]) – List of [search energy, search width] search parameters.

Return type:

tuple[list[dict], list[dict], list[dict]]

Returns:

Tuple of results, where index 0 contains all matches, index 1 primary matches only, and index 2 secondary matches only. The elements in the tuples are lists of dictionaries.

Dictionary keys

  • element: the element which was searched for

  • energy: the energy which was searched for

  • error: the error range used to search within (takes values of 1x search width, 2x search width or 3x search width)

  • peak_centre: energy of transition matched in database

  • transition: name of transition matched in database

  • diff: difference between searched energy and match (how close the match is)

EVA.core.data_searching.get_match.search_muxrays_single_transition(input_element, input_trans)

Searches in muonic xray database for all peaks specified for a single transition. When using the Mudirac database, there will always be only one database entry per transition. However, the legacy database can sometimes contain multiple entries for a single transition. Hence, this function returns a list of dictionaries rather than a single.

Parameters:

  • input_element (str) – which element to search for

  • input_trans (str) – name of transition to search for (in spectroscopic notation e.g. (2p3/2->1s1/2))

Return type:

list[dict]

Returns:

List of dictionaries, one for each match, with keys

  • element (str)

  • energy (float)

  • transition (str)
