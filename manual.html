Loading data

To load your data into EVA, first go to File>Browse to Data Directory and select the folder containing your data. EVA will then automatically load one of your runs. You can choose which run to load by editing the Run Number field in the center of the control panel. Press the load button to open the peak analysis window.

Using the peak analysis tool

The peak analysis window can be used to identify peaks within your data. You can click on different peaks in your data to search for possible sources. Left-click on a peak to search for possible muonic x-ray transitions, or right-click to search for gamma transitions.
